**CodeAlpha Python Internship Project**

This repository contains Python projects completed as part of the CodeAlpha Python Programming Internship.

Repository Name
**CodeAlpha_Project**

**Projects Included**

**1. Basic Chatbot**
A simple rule-based chatbot that responds to user input using predefined conditions.

**Features:**
- Responds to inputs like "hello", "how are you", "bye"
- Uses if-elif conditions
- Runs in a loop until exit command
- Beginner-friendly console application

 **2. Hangman Game**
 A text-based Hangman game where the user guesses a hidden word one letter at a time.

 **Features:**
- 5 predefined words
- Maximum 6 incorrect attempts allowed
- Uses loops, strings, lists, and conditionals
- Fully console-based game


 **3. Stock Portfolio Tracker**
 A simple Python program that calculates total investment based on user input and stock prices.

 **Features:**
- Uses a dictionary for stock prices
- User inputs stock name and quantity
- Calculates total investment value
- Basic arithmetic operations

 **Technologies Used**
- Python 3
- Core Python concepts:
  - Loops
  - Functions
  - Dictionaries
  - Conditional statements
  - Strings
  - Input/Output handling
